```
┌─────────────────────────────────────────────────────────────────────────────────────────┐
│                                     关键配置                                             │
├─────────────────────────────────────────────────────────────────────────────────────────┤
│                                                                                         │
│  环境变量 (.env)                                                                        │
│  ├── SERPAPI_KEY          # SerpAPI 密钥                                               │
│  ├── OPENAI_API_KEY       # LLM API 密钥 (兼容 OpenAI 格式)                             │
│  ├── OPENAI_API_BASE      # LLM API 基础 URL                                           │
│  └── KB_MODEL_DEFAULT     # 默认使用的模型名称                                          │
│                                                                                         │
│  类型注册表 (extractors/type_registry.py)                                               │
│  ├── 知识类型定义                                                                       │
│  ├── 类型简介 (用于路由)                                                                │
│  ├── Schema 路径                                                                        │
│  └── Prompt 路径                                                                        │
│                                                                                         │
│  专用抽取器 (extractors/se_kb_extractors.py)                                            │
│  ├── 每种知识类型对应一个抽取器类                                                        │
│  ├── 定制化的系统 Prompt                                                                │
│  └── 预期输出结构说明                                                                   │
│                                                                                         │
└─────────────────────────────────────────────────────────────────────────────────────────┘
```

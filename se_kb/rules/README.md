# 📘 rules/ — 高层建模原则（跨图通用）

此目录定义了所有建模活动中**跨图通用、与图类型无关**的高层规则。

这些规则指导“如何正确建模”，在任何图类型中都适用。

## 包含内容

- **naming.json**  
  跨图统一命名原则：动词—名词结构、数据项命名、流程术语规范。

- **quality.json**  
  高质量建模标准，包括：一致性、完整性、可追踪性、可读性等。

- **decomposition.json**  
  分解原则：自顶向下、3–7 原则、层次化结构、扇入扇出约束。

## 适用场景
- 作为建模风格的统一标准  
- 所有 diagrams 的 rules 在此基础上做特化  
- Agent 在抽取内容时确保不同图之间具有统一风格
